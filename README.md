# Автоматизированные тесты для DummyJSON API

Проект содержит автоматизированные тесты для проверки API сервиса [DummyJSON](https://dummyjson.com). Тесты написаны на Python с использованием библиотек Pytest, HTTPX и Pydantic. Для генерации тестовых данных применяется Faker, а для формирования отчетов — Allure.

## Стек технологий
* **Python** — основной язык программирования
* **Pytest** — фреймворк для тестирования
* **HTTPX** — клиент для выполнения HTTP-запросов
* **Pydantic** — валидация схем ответов API
* **Faker** — генерация фейковых данных
* **Allure** — фреймворк для создания отчетов о тестировании

## Структура проекта
* `src/` — исходный код проекта (API клиенты, схемы Pydantic, утилиты).
* `tests/` — тестовые сценарии.
* `config.py` — конфигурация и настройки тестовой среды.
* `conftest.py` — фикстуры Pytest.
* `pytest.ini` — настройки Pytest по умолчанию.
* `requirements.txt` — зависимости проекта.

## Требования
Убедитесь, что у вас установлены:
* Python 3.10 или выше
* Allure Commandline (требуется для генерации и просмотра HTML-отчетов)

## Инструкция по установке и настройке

1. **Склонируйте репозиторий** (или перейдите в папку проекта):
   ```bash
   git clone https://github.com/scy7he-edu/api_tests
   cd api_tests
   ```

2. **Создайте виртуальное окружение:**
   ```bash
   python -m venv venv
   ```

3. **Активируйте виртуальное окружение:**
   * Для Linux/macOS:
     ```bash
     source venv/bin/activate
     ```
   * Для Windows:
     ```bash
     venv\Scripts\activate
     ```

4. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

## Запуск тестов

Для запуска всех тестов выполните команду:
```bash
pytest
```
*Конфигурация в `pytest.ini` уже настроена на генерацию Allure-результатов в папку `allure-results`.*

Для запуска конкретных тестов по маркерам можно использовать:
```bash
pytest -m positive  # запуск только позитивных тестов
pytest -m negative  # запуск только негативных тестов
```

## Просмотр отчетов Allure

После прогона тестов сформируется папка `allure-results`. Чтобы посмотреть красивый HTML-отчет, выполните команду:
```bash
allure serve allure-results
```
Эта команда автоматически сгенерирует отчет и откроет его в вашем браузере по умолчанию.
